# MEMORY

## Product context

Shipping Label Helper is a no-login-first-value utility site for ecommerce sellers who need to fix shipping label size, scale, paper, carrier, marketplace, and printer setup problems.

## SEO operating principles

- Every SEO page should map to a real user task, not a thin keyword page.
- Pages should lead back to a product action, tool action, content cluster, or conversion action.
- Prefer technical SEO basics first: metadata, canonical, sitemap, robots, schema, FAQ, internal links, OG/Twitter metadata, and updated dates.
- Avoid claims such as official, certified, guaranteed, or carrier-approved unless independently verified.
- Search Console / Analytics / AdSense decisions require real data access.

## Current implementation notes

- Next.js App Router project using Next 16 and next-intl.
- Production domain is `https://labelhelper.com`.
- Programmatic SEO content is centralized in `data/seo-pages.ts` and route mapping in `lib/seo-route-map.ts`.
- Metadata helpers live in `lib/seo.ts`.
- Sitemap and robots are generated by `app/sitemap.ts` and `app/robots.ts`.
